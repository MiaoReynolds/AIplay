## Project Introduction
AIPlay focuses on **AI application exploration + practical building**.
The current approach is to reproduce useful open-source ideas, turn them into working prototypes, and then iterate with custom extensions.

Main exploration directions:
- Desktop screen gadgets
- LED and embedded mini devices
- Robot hardware (arm, quadruped, control systems)
- AI workflow automation and agent tools
- AI learning notes and knowledge organization

- **Prompt Generator**: Utility page for generating and refining prompts.  
  Link: [Prompt Generator](https://miaoreynolds.github.io/AIplay/prompt-generator/)
