# Quadruped Robot

An experimental AI-assisted walking robot:

| Module      | Description                    |
|-------------|--------------------------------|
| Locomotion  | 4-legged servo structure       |
| Brain       | Microcontroller + IMU          |
| Goals       | Stability, gait learning       |
