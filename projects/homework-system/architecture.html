Homework Assistant — System Architecture

AI-powered homework supervision system · Voice · Vision · Telegram

Frontend — Child's UI

  • Three-column responsive layout
  • Subject cards with status badges
  • Camera capture + crop for submissions
  • Chat panel (voice + text)
  • "Ask AI" photo tutoring panel
  • Review panel with AI grading results
  • Edgeless fullscreen on Ubuntu kiosk

Backend — Flask Server

  • REST API on port 8791
  • Photo submission with date-named files
  • Status tracking (pending → submitted → reviewed)
  • Edge TTS voice generation (per-subject voice)
  • ffplay audio output (one-at-a-time)
  • Homework logbook auto-generation
  • Chat history (JSON + Markdown)

Automation — Cron Jobs

  • 14:15 — Kickoff + voice reminder
  • every 3 min — Fast grading via Gemini
  • every 5 min — Chat auto-responder
  • every 15 min — Forward to parent Telegram
  • 15:00 / 15:45 / 16:30 — Mid-check + grading
  • 17:30 — Final check + summary report
  • Parents send messages via prefix "告诉孩子"
